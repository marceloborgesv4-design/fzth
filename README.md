# From Zero to Hero — AWS Partner Program (EN)

English version of the FZTH landing page (translated from Spanish), keeping the
original layout, colors and structure.

## Contents
- `index.html` — the landing page (self-contained, English)
- `assets/` — official AWS logo (white variant) + source SVG
- `FZTH_Landing_EN.pdf` — downloadable PDF version of the page

## Publishing (GitHub Pages)
Deploy from the `main` branch, root folder. The site is served at
`https://<user>.github.io/<repo>/`.
